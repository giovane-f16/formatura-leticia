# User Flows

## Fluxo 1 - Ver Homenagem

1. Usuario abre a Home.
2. Usuario le a mensagem principal.
3. Usuario visualiza fotos de destaque.
4. Usuario acessa a galeria ou o jogo pela navegacao.

## Fluxo 2 - Navegar Pela Galeria

1. Usuario entra na pagina de fotos.
2. Usuario visualiza a grade responsiva.
3. Usuario le titulos e textos curtos das fotos.
4. Usuario volta para a Home ou acessa o jogo.

## Fluxo 3 - Jogar

1. Usuario entra na pagina de jogo.
2. Usuario ve a pergunta "Voce me ama?".
3. Usuario tenta interagir com "Nao" ou escolhe "Sim".
4. Ao escolher "Sim", ve a segunda pergunta.
5. Usuario escolhe "Sim" na pergunta do carro.
6. Usuario ve o GIF final.

## Fluxo 4 - Editar Conteudo

1. Desenvolvedor abre o arquivo de dados do projeto.
2. Desenvolvedor altera textos, imagens, titulos e descricoes.
3. Interface reflete as mudancas sem alterar componentes estruturais.
